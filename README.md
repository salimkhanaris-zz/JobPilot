# 🚀 JobPilot

> Local-first AI-powered desktop application for intelligent job search and application automation.

## Overview

JobPilot helps job seekers automate repetitive application tasks while keeping them in complete control.

It can:

- 📄 Manage multiple resumes
- 🤖 Match resumes to job descriptions using AI
- 🌐 Automate browser-based applications
- 📝 Generate tailored cover letters
- 📊 Track applications and interviews
- 🔌 Support multiple job portals through plugins

## Features (Planned)

- Resume Library
- Job Queue
- AI Resume Matching
- Browser Automation
- Application Tracking
- Cover Letter Generation
- Plugin System
- Local & Cloud AI Support

## Technology Stack

- Frontend: React + Tauri
- Backend: Python
- AI: Ollama (Local) + Optional Cloud Providers
- Browser Automation: Playwright
- Database: SQLite
